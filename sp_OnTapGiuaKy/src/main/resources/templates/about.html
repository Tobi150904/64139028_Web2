Giới thiệu về trang web

Trang web này được xây dựng nhằm mục đích quản lý thông tin sinh viên.

Các chức năng chính bao gồm:

  • Xem danh sách sinh viên
  • Thông tin liên hệ
  • Ví dụ về hashCode trong Java

Mục tiêu

Trang web được phát triển để minh họa cách sử dụng Spring Boot kết hợp với Thymeleaf để tạo ứng dụng web với các thành phần tái sử dụng.
